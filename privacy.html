Privacy Policy

Last updated: March 1, 2026

Fake Video Detector for Instagram is a browser extension that helps users identify AI-generated videos on Instagram. We take your privacy seriously.

Data We Collect

We do not collect, transmit, or store any personal data on external servers. All data generated by this extension stays on your device.

What Is Stored Locally

The following data is stored in your browser's local extension storage:

  • API key — Your Google Gemini API key, used to send video frames for analysis. This key is stored locally and never sent to any server other than Google's Gemini API.
  • Cached analysis results — Verdicts for previously analyzed videos, to avoid redundant API calls.
  • User feedback labels — If you label a video as "AI" or "Real," that label is stored locally for potential training data export.
  • Usage statistics — Daily counts of videos analyzed and API calls made, stored locally.
  • Error logs — The last 20 error messages for debugging, stored locally.
  • Settings — Your auto-analyze preference.

Third-Party Services

When you provide a Gemini API key and auto-analysis is enabled, the extension sends video frame data (as images) and publicly visible Instagram comments directly to the Google Gemini API (generativelanguage.googleapis.com) using your own API key. This communication is between your browser and Google's servers — no data passes through our servers.

Please refer to Google's AI Terms of Service for how Google handles data sent to the Gemini API.

No Analytics or Tracking

This extension does not include any analytics, telemetry, or tracking. There are no cookies, fingerprinting, or user identification mechanisms.

Data Export and Deletion

You can export all locally stored feedback data using the "Export Training Data" button in the extension popup. You can delete all extension data by removing the extension from your browser or clearing its storage via chrome://extensions.

Changes to This Policy

If this policy is updated, the "Last updated" date at the top will be revised. Continued use of the extension after changes constitutes acceptance of the updated policy.
